# NNUtils
This package provides neural net utilities:
- standard networks
  - [x] mobilenets
  - [ ] resnet
- neural network blocks
  - [x] DepthwiseSeparableConv
  - [x] BottleneckResidual
  - [x] ResnetResidualv1
  - [x] ResnetResidualv2
- neural network layers
  - [x] Sinc Conv
- saliency maps 
  - [x] Gradient
  - [x] SmoothGradient
  
based on [Flux.jl](https://github.com/FluxML/Flux.jl).
